# Aoc_2017
Trying to learn elixir while doing AoC 2017

## Installation

  1. Add aoc_2017 to your list of dependencies in mix.exs:

        def deps do
          [{:aoc_2017, "~> 0.0.1"}]
        end

  2. Ensure aoc_2017 is started before your application:

        def application do
          [applications: [:aoc_2017]]
        end
